COURSE: LEARN JAVA

2D Arrays: Image Manipulation Project

Project description: In this project, you will be creating an application which is able to modify images as well as create new images using 2D arrays! The first section covers stretching the image horizontally, shrinking the image vertically, negating the color, applying a color filter, and inverting the image. The second section covers creating an image consisting of random pixels, placing a rectangle in the image, and using the method to randomly place many rectangles in the image.

COURSE: BUILDING INTERACTIVE JAVASCRIPT WEBSITES

Musicon

Project description: In this project, you will be updating the website of an online musical instruments store, Musicon. You will incorporate your knowledge of HTML, CSS, JavaScript, and Handlebars to make a multi-page interactive website! Your job is to create a navigation bar using unordered lists, anchor tags, and class attributes, style the website, build out semantic templates using Handlebars, and display relevant information using JavaScript objects, arrays, and control flow alongside built-in Handlebars iteration and control flow helpers.

In this project, you'll use your knowledge of HTTP requests and asynchronous JavaScript to create a movie discovery app that will recommend random movies by genre. COURSE: LEARN NODE.JS

Rock-Paper-Scissors

Project description: Using your knowledge in Node.js, let's create an online version of Rock-Paper-Scissors where you can play against the HTTP server. Using the http module, we will build a simple HTTP server. We will practice how to route requests based on pathname and request methods. We will also be using other core Node.js components in this project—we will use the fs module to stream HTML files and create an URL object to extract queries from a URL.

COURSE: LEARN REACT

Passing Thoughts

Project description: What if you could post something and know it wouldn't live forever? In this project, we'll build a place for our passing thoughts. Once you add a short thought, it'll disappear after just 15 seconds.

COURSE: LEARN SPRING

The Boots Bootique

Project description: A new shop, the Boots Bootique, is set to open their flagship store this spring. However, they've got no way to manage their inventory yet! The store owns a collection of boots of different sizes, types, materials, and quantities. Help the shop create their inventorying system in time for their opening. For this project, we'll use Spring Data JPA to complete the functionality for the Boots API so the frontend can: Create new boots, Read and search the boots in the inventory, Update the number of any given boot in the inventory, Delete boots from the inventory.
